# 📊 Portafolio de Análisis de Datos, SQL y Python

¡Hola! Soy **Narciso Brian García Texis**, profesional en Contaduría Pública e Ingeniería en Desarrollo de Software. Este repositorio reúne proyectos y simulaciones de análisis de datos, optimización de consultas SQL en entornos empresariales (Oracle EBS) y automatización con Python.

## 🛠️ Habilidades Clave
* **Bases de Datos & Análisis:** SQL / PL-SQL (Oracle 11g), ANSI JOINs, depuración de reportes legados, análisis de nóminas y finanzas.
* **Lenguajes & Automatización:** Python (Pandas, NumPy, oracledb, validación de estructuras XML/CFDI).
* **Business Intelligence & Herramientas:** Excel Avanzado, Power BI, Tableau, Toad, Spyder/PyCharm.

---

## 📁 Proyectos Destacados (Simulaciones & Sanitizados)

### 🔹 1. Optimización y Depuración de Reportes SQL (Oracle EBS)
* **Descripción:** Reestructuración de consultas SQL legadas para resolver errores de agregación duplicada (`ORA-00937`) e *implicit joins*, migrando la sintaxis a **ANSI JOINs**.
* **Impacto:** Reducción del tiempo de ejecución en un **82%** en reportes críticos de Nómina/RH y ampliación de cobertura a contratos de honorarios.

### 🔹 2. Validador y Extractor de CFDI / XML en Python
* **Descripción:** Desarrollo de un script automatizado para la lectura, extracción y validación de estructuras XML de recibos de nómina frente a especificaciones del SAT.
* **Impacto:** Detección temprana de inconsistencias antes del timbrado definitivo.

### 🔹 3. Análisis y Depuración de Datos de Pago
* **Descripción:** Modelado y limpieza de bases de datos para conciliación de pólizas contables y flujos de pago.
* **Impacto:** Reducción del 35% en errores operativos de procesamiento.
